# Project Overview

## Title
SUPGS – Sustainable Urban Planning using Green Systems

## Idea
This project studies how natural green systems and optimized road planning
can reduce CO₂ emissions in urban environments.

## Objective
- Reduce carbon emission by 10–15%
- Use natural processes instead of expensive artificial systems
- Provide scalable urban planning model

## Motivation
Rapid vehicle growth increases CO₂ levels.
This project proposes a sustainable and realistic mitigation approach.
